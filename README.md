# 🛍️ Myntra Product Catalog Intelligence Dashboard

## 📌 Project Overview
An end-to-end data analytics project analyzing 527,000+ 
Myntra fashion product listings to uncover pricing strategy, 
discount patterns, category distribution, and gender assortment insights.

> ⚠️ This is a Product Catalog dataset — not a sales/transactions 
> dataset. Metrics represent listed products, not actual orders.

---

## 🔧 Tech Stack
| Tool | Purpose |
|------|---------|
| Python (Pandas) | Data cleaning & preprocessing |
| SQL | Exploratory data analysis & querying |
| Power BI | Interactive dashboard |
| DAX | Calculated measures & KPIs |

---

## 📊 Key Metrics
- 🛒 Total Products: 527K listings
- 💰 Total MRP Value: ₹1.27bn
- 🏷️ Total Discounts Offered: ₹466.31M
- 📦 Catalog Value (Discounted): ₹804.85M
- ⭐ Total Customer Reviews: 12M
- 📉 Avg Discount Rate: ~36.7%

---

## 💡 Key Insights
- Indian Wear is the largest category with 146K products
- Zotw brand offers the highest avg discount at 63%
- 64% of catalog targets Women vs 36% Men
- Western wear generates the highest catalog value
- Most products are highly rated (4.5 stars dominant)
